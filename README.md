# AI Plagiarism Checker

A plagiarism detection web application built using Python, Streamlit, TF-IDF, and Cosine Similarity.

## Features

- Upload multiple text (.txt) files
- Preprocesses text automatically
- Detects similar documents using TF-IDF
- Calculates cosine similarity
- Generates a plagiarism report
- Download report as CSV

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- NLP
- TF-IDF
- Cosine Similarity

## Project Structure

AI-Plagiarism-Checker/
├── app.py

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
